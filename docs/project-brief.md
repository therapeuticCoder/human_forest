# Project Brief

## Project name

Calm React Starter

## Purpose

Provide a clean, reusable React starter for small, agent-friendly front-end applications.

## Intended users

- The human project owner
- Coding agents working under human supervision
- Future projects that need a calm, generic React foundation

## Scope

This starter includes:

- React + TypeScript + Vite
- Tailwind CSS
- shadcn/ui
- Vitest
- basic layout components
- agent-ready repo files and workflow docs

## Out of scope

This starter does not include:

- domain-specific tools
- therapy content
- clinic website content
- adulting/fantasy app features
- authentication
- analytics
- backend services
- database configuration

## Success criteria

This starter is successful when a new project can inherit it and immediately have:

- a working React app
- a clean UI foundation
- clear agent instructions
- a local backlog
- recorded decisions
- a basic human-agent workflow
- passing checks
